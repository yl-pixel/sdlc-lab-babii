# sdlc-lab-babii
Трекер води — додавання випитих склянок води протягом дня.

## 1. Планування
Мета: створити простий додаток **Water tracker**, який дозволяє користувачам моніторити випитих склянок води протягом дня.

## 2. Аналіз вимог (User stories)
1. Як користувач, я хочу додавати випиті склянки води протягом дня. ✅ (Must have)
2. Як користувач, я хочу бачити статистику за певний проміжок часу. ✅ (Must have)
3. Як користувач, я хочу рекомендації та поради для мого вживання води.
4. Як користувач, я хочу мати різні види води та бачити їхній склад через додаток.
5. Як користувач, я хочу отримувати нагадування про вживання води.

## 4. Реалізація (Псевдокод)
```pseudo
function addGlass():
	currentGlasses++
	updateProgressBar(currentGlasses / dailyGoal)
	if (currentGlasses > dailyGoal) {
		showMessage("Ціль за день вже була досягнута")
	}	else showMessage("Вам ще залишилось %i склянок води до кінця дня", currentGlasses)
```
## 5. Тестування
1. Додади склянку випитої води → прогрес змінюється.
2. Видалити склянки випитої води → кількість випитих склянок води може бути від'ємною, помилка.
3. Досягнута ціль за день → показує повідомлення про її досягнення.

## 6. Висновки
Для цього простого проекту найкраще підійде **Agile-підхід** із короткими ітераціями, поступово покращуючи додаток.
Він підійде для швидкого додавання нових функцій та для швидкого адаптування продукту під фідбек користувачів.
Waterfall - повільний для проєкту, а Spiral — складний.
